👋 환영합니다 - 재고관리 대시보드

이 대시보드는 제품을 만들 때 사용되는 재료의 재고를 자동으로 관리하기 위한 시스템입니다. 제품 생산 수량만 입력하면 매핑된 재료가 알아서 차감되고, 부족한 재료는 빨간 배지로 알려드립니다.

💾 모든 데이터는 본인 구글시트에 저장됩니다. 이 페이지는 시트를 읽고 쓰는 창일 뿐입니다.

빠른 이동:

처음 한 번 재료 관리 제품 & BOM 생산 입력 입고/조정 취소/이력 알림 이해하기 자주 묻는 질문

0처음 한 번만 하면 되는 일

아래 3단계 순서대로만 등록해두면 이후로는 [생산 입력] 한 화면만 쓰면 됩니다.

  1. 재료 등록 — 가지고 있는 모든 재료를 [재료 현황] 탭에 추가
  2. 제품 등록 + BOM 매핑 — [제품 & BOM] 탭에서 제품을 만들고, 그 제품 1개를 만들 때 들어가는 재료들을 지정
  3. 생산 시작 — 이후로는 [생산 입력] 탭에서 "어떤 제품을 몇 개 만들었다"만 입력

1재료 관리

[재료 현황] 탭 → + 재료 추가 클릭 → 항목을 채우고 [추가].

입력 항목 의미:

  • 재료명: 로제트 리본, 키링고리 같이 한눈에 알아볼 이름
  • 단위: 셀 수 있는 거면 개, 길이는 m 또는 cm, 무게는 g
  • 현재재고: 지금 가지고 있는 수량
  • 임계수량: 현재재고가 이 값 미만으로 떨어지면 빨간 알림이 뜹니다. 단위는 그 재료의 단위와 동일 (m, 개, g 등)
  • 메모: 색상/규격/공급처 등 자유 메모
  • 사진: 한눈에 알아보기 위한 대표 이미지 (선택)
예시: 로제트 리본 / 단위 m / 현재재고 100 / 임계수량 20
→ 재고가 20m 미만으로 떨어지면 부족 빨간 알림이 표시됩니다.

사진 관리:

  • 재료 추가 모달에서 [파일 선택]으로 PC에서 업로드 (자동으로 800px로 줄여 저장됨)
  • 이미 등록된 재료는 테이블의 + 박스 클릭으로 추가
  • 섬네일 클릭 → 큰 미리보기에서 [교체] / [삭제] 가능
  • 사진은 본인 구글드라이브의 재고관리_이미지 폴더에 저장됨

현재재고 / 임계수량 수정: 테이블 셀을 직접 클릭해서 숫자를 바꾸고 다른 곳을 클릭하면 자동 저장됩니다.

💡 임계수량은 "이 정도까지 떨어지면 발주해야 한다"는 안전 라인으로 정하세요. 발주 리드타임(주문 → 도착)이 길수록 임계수량을 크게 잡는 게 안전합니다.

2제품 & BOM 등록

BOM (Bill of Materials)은 자재명세서입니다. 제품 1개를 만들 때 들어가는 재료의 목록과 각각의 소요량이에요.

예시 — 로제트 키링 1개의 BOM
· 로제트 리본 0.3m
· 키링고리 1개
· 글루건 스틱 0.5g

등록 방법:

  1. [제품 & BOM] 탭 → + 제품 추가로 제품을 먼저 생성
  2. 왼쪽 제품 목록에서 방금 만든 제품 클릭
  3. 오른쪽 BOM 영역에서 + 재료 추가로 재료를 하나씩 추가하고 1개당 소요량 입력
  4. BOM 저장 클릭
💡 한 재료가 여러 제품에 공통으로 들어가도 됩니다. 각 제품마다 BOM에 같은 재료를 추가하기만 하면 됩니다.

제품 종류 예시: 같은 로제트 키링이라도 "기본형"과 "글리터 추가형"을 별도 제품으로 등록하면 생산 시 BOM 차이가 자동으로 반영됩니다.

3생산 기록하기 (가장 자주 쓸 기능)

매일 가장 많이 사용하게 될 화면입니다. [생산 입력] 탭에서:

  1. 드롭다운에서 만든 제품 선택
  2. 생산 수량 입력 (예: 10)
  3. 아래의 예상 차감 미리보기에서 어떤 재료가 얼마씩 빠질지 확인
  4. 괜찮으면 생산 기록 추가 클릭

→ BOM에 매핑된 모든 재료가 (1개당 소요량 × 생산수량)만큼 자동 차감되고, [재료 현황]의 수량이 업데이트됩니다.

⚠ 재료가 부족할 경우 차감 시도 시 경고가 뜹니다. 그래도 강행하면 마이너스 재고로 기록되니, 보통은 먼저 입고를 등록한 뒤 생산을 입력하세요.

4입고 / 조정

재료가 새로 들어왔거나 실물과 시스템 수량이 안 맞을 때 [입고/조정] 탭에서 기록합니다.

  • 입고 (재료 도착): 변동 수량을 양수(+)로, 종류는 입고
  • 조정 (분실/파손/오차 보정): 변동 수량을 음수(-)로, 종류는 조정
예시: 리본 50m 입고 → 재료 선택 로제트 리본, 변동 수량 +50, 종류 입고, 메모 A사 발주분

5잘못 입력했을 때 (이력 & 취소)

실수로 잘못 입력했다면 [이력] 탭으로 가서 해당 줄의 취소 버튼을 누르면 됩니다.

  • 생산 기록을 취소하면 → 차감되었던 재료가 자동 복원
  • 입고 기록을 취소하면 → 늘어났던 재료가 자동 감소
  • 취소된 기록은 회색 취소선으로 표시되며 시트에는 남아 있음 (감사 추적용)

날짜별 조회는 상단의 최근 7일/30일/90일/전체 필터로 가능합니다.

6알림 읽는 법

상단 알림 바와 재료 테이블 행에 표시되는 배지의 의미:

  • 🔴 부족 — 현재재고 < 임계수량. 즉시 발주 필요.
  • ⚠ 주의 — 현재재고 < 임계수량 × 2. 슬슬 발주 고려.
  • ✓ 정상 — 임계수량의 2배 이상. 충분히 여유 있음.
  • 소진 — 재고 0 이하. 생산 불가.
  • 미설정 — 임계수량이 0이거나 미입력. 알림 비활성화.
예: 단위 m, 현재재고 15, 임계수량 20 → 15 < 20 이므로 부족
예: 현재재고 30, 임계수량 20 → 20 ≤ 30 < 40 이므로 주의
예: 현재재고 50, 임계수량 20 → 50 ≥ 40 이므로 정상

❓ 자주 묻는 질문

Q. 다른 분이 대시보드를 보려면?
A. dashboard.html 파일을 공유 폴더(드롭박스/구글드라이브/원드라이브)에 두거나 깃허브 페이지에 호스팅하면 같은 URL로 접근할 수 있습니다. 처음 한 번 [설정] 탭에 동일한 웹앱 URL을 입력하면 그 사람의 브라우저에 저장됩니다.
Q. 사진은 어디에 저장되나요?
A. 본인 구글드라이브 최상위의 재고관리_이미지 폴더에 자동 저장됩니다. 폴더는 첫 업로드 때 만들어집니다.
Q. 구글시트의 데이터를 직접 수정해도 되나요?
A. Materials, Products, BOM, Settings 시트는 직접 수정해도 됩니다 (긴급 시). 단 ProductionLog, StockLog는 대시보드에서만 추가/취소하세요 — 재고 자동 차감/복원 로직과 짝이 안 맞으면 수치가 어긋날 수 있습니다.
Q. 실수로 시트의 데이터를 잘못 건드렸어요.
A. 구글시트는 모든 편집 이력이 자동 저장됩니다. 시트 상단 메뉴 파일 → 버전 기록 → 버전 기록 보기에서 이전 시점으로 복원할 수 있습니다.
Q. 화면에 최신 데이터가 안 보입니다.
A. 우상단 🔄 새로고침 버튼을 누르면 시트에서 최신 데이터를 다시 가져옵니다. 시트를 다른 사람이 동시에 수정 중이라면 변경 후 새로고침해야 보입니다.
Q. 임계수량은 모든 재료가 같아야 하나요?
A. 아니요. 재료마다 다릅니다. [재료 현황] 테이블의 임계수량 셀을 직접 수정하면 그 재료에만 적용됩니다. 신규 추가 시 기본값은 [설정] 탭에서 조정 가능합니다.
Q. 임계수량은 어떻게 정하면 좋나요?
A. 발주 → 도착까지 걸리는 일수 × 일평균 소비량 정도가 안전한 기준입니다. 예: 리본을 평균 하루 5m 쓰고 발주 도착에 3일 걸리면, 임계수량을 15~20m 정도로 두면 재고가 떨어지기 전에 알림을 받고 발주할 수 있습니다.
Q. 한 재료가 여러 제품에 들어가요. 따로 등록해야 하나요?
A. 재료는 한 번만 등록하면 됩니다. 각 제품의 BOM에 같은 재료를 추가하기만 하면, 어떤 제품을 생산하든 그 재료가 공통으로 차감됩니다.
Q. 재료를 삭제하려는데 "BOM에서 사용 중"이라고 나옵니다.
A. 그 재료를 사용하는 제품의 BOM에서 먼저 제거하거나, 삭제 시도 후 뜨는 "강제 삭제" 옵션을 선택하면 됩니다. (강제 삭제 시 BOM이 참조하는 재료가 빠진 상태로 남으니 주의)
Q. 시트의 컬럼 순서나 이름을 바꿔도 되나요?
A. 안 됩니다. 시트의 2행 헤더 이름(예: id, 재료명, 현재재고...)을 그대로 유지해야 대시보드가 인식합니다.
Q. Apps Script를 수정하면 어떻게 다시 적용되나요?
A. Apps Script 편집기에서 코드 저장 후 배포 → 배포 관리 → 연필 ✏️ → 버전: 새 버전 → 배포를 다시 하면 적용됩니다. 기존 웹앱 URL은 그대로 유지됩니다.

📦 데이터 백업

구글시트 자체가 백업입니다. 추가로 안심하려면:

  • 버전 기록: 시트는 모든 편집 시점 자동 저장 (파일 → 버전 기록)
  • 주기적 사본: 시트 메뉴 파일 → 사본 만들기로 월말마다 사본 저장 권장
  • 다운로드: 파일 → 다운로드 → Microsoft Excel(.xlsx)로 PC에 받아두기
