# Stock System

로제트 키링 재료 재고 관리 시스템.
Google Sheets를 데이터베이스로 사용하는 단일 HTML 대시보드.

## 주요 기능

- 재료 마스터 관리 (사진 포함, 셀 직접 편집)
- 제품 ↔ 재료 BOM (자재명세서) 매핑
- 제품 생산 입력 시 매핑된 재료 자동 차감
- 재료 입고 / 수동 조정 기록
- 임계수량 미만 시 부족 알림 (재료별 개별 설정)
- 생산/입고 이력 + 잘못 입력 시 [취소] 기능
- 화면 내 ❓ 사용 가이드 탭

## 구성

| 파일 | 역할 |
|---|---|
| `dashboard.html` | 단일 페이지 대시보드. 브라우저로 열기. |
| `apps_script.gs` | Google Apps Script 백엔드 코드. 구글시트에 붙여넣기. |
| `재고관리_템플릿.xlsx` | 구글시트로 업로드해서 사용할 초기 양식. |

## 아키텍처

```
[브라우저 dashboard.html]
        ↓  POST JSON
[Google Apps Script Web App]
        ↓  read/write
[Google Sheets (6개 시트: Materials/Products/BOM/ProductionLog/StockLog/Settings)]
        +
[Google Drive: 재고관리_이미지 폴더]
```

## 설치 (5단계, 약 15분)

1. **구글시트 생성**
   - `재고관리_템플릿.xlsx` 를 구글드라이브에 업로드
   - 우클릭 → 연결 앱 → Google Sheets

2. **Apps Script 백엔드 설치**
   - 시트의 메뉴에서 `확장 프로그램 → Apps Script`
   - `apps_script.gs` 내용 전체 복사해서 붙여넣기 → 저장

3. **웹 앱 배포**
   - 우상단 `배포 → 새 배포` → 유형: **웹 앱**
   - 실행: 나, 액세스: 누구나 (또는 조직 내)
   - 웹 앱 URL 복사

4. **대시보드 실행**
   - `dashboard.html` 더블클릭 (브라우저에 열림)
   - 첫 화면 [초기 설정]에 웹 앱 URL 붙여넣고 [저장 & 연결]

5. **첫 사용**
   - 재료 등록 → 제품 & BOM 매핑 → 생산 입력 → 끝

## 사용 가이드

대시보드 상단의 **❓ 사용 가이드** 탭에 처음 사용자도 따라할 수 있는 단계별 안내가 포함되어 있습니다.

## 데이터 흐름 예시

```
[재료 등록]   리본 100m, 임계수량 20m
[제품 BOM]    로제트 키링 1개 = 리본 0.3m + 키링고리 1개 + 글루건 0.5g
[생산 입력]   로제트 키링 10개 생산
              → 자동: 리본 -3m, 키링고리 -10개, 글루건 -5g
[알림 표시]   리본 잔량 17m < 임계 20m → 🔴 부족 배지
[입고 입력]   리본 +50m 도착
              → 잔량 67m → ✓ 정상
```

## 보안 참고

- 대시보드 HTML에는 URL/시크릿이 하드코딩되어 있지 않습니다 (각자 브라우저 localStorage에 저장).
- Apps Script 웹 앱 URL은 알아낼 수 없으면 접근 불가하지만, 민감 데이터를 다룬다면 액세스 권한을 "조직 내"로 제한하세요.
